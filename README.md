## Specs

1. Add Users

## Setup
#### First clone this repo:
```
git clone https://github.com/ekibet/passwordLocker.git
cd PasswordLocker
```
#### Install requirements
```pip install -r requirements.txt```

#### You can now run the code
```
python run.py
```
##### or 
```
python gui.py
```
